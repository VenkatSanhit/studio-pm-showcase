# Studio PM Showcase
    
    A movie-themed portfolio website showcasing various features related to films.
    
    ## Features
    
    * Opening Credits
    * Metrics Board
    * Feature Films Showcase
    * Behind The Scenes
    * Awards and Accolades
    * Coming Soon Section
    * Contact Information
    * AI Chat Integration
    
    ## Technologies Used
    
    * Vite
    * TypeScript
    * React
    * shadcn-ui
    * Tailwind CSS
